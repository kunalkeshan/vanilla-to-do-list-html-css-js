How to Use?

  • Use the add task input to add new tasks.
  • Search Existing Tasks with the search Task Input.
  • Simply click On the task to mark as completed. The Text will be strikethrough.
  • Click on the trash Icon to delete the task.
  • Click on the Edit task Icon to edit task name.

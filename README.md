<center><h1>To-Do List</h1></center>
<p> A simple to do list, made with vanilla HTML, CSS and JS.</p>
<h2>Features</h2>
<ul>
  <li>Create new tasks.</li>
  <li>Edit an already existing task</li>
  <li>Delete a task</li>

  <li>Search in between tasks</li>
</ul>
